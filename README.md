# 🎰 카지노 게임 - 포커 & 블랙잭 & 슬롯머신

웹 기반 카지노 게임 모음입니다. 텍사스 홀덤 포커, 블랙잭, 다이아몬드 슬롯머신을 즐길 수 있습니다!

## 🃏 포커 게임 규칙

- **시작 칩**: 플레이어와 AI 모두 1000칩으로 시작
- **블라인드**: 스몰 블라인드 10, 빅 블라인드 20
- **승리 조건**: 상대방의 모든 칩을 획득
- **패배 조건**: 모든 칩을 잃음
- **AI 특징**: 공격적인 플레이 스타일, 블러핑 사용, 큰 베팅도 받아줌

## 🎮 포커 게임 방법

1. "포커" 버튼을 클릭하여 게임 시작
2. 각 라운드마다 2장의 홀 카드를 받음
3. 베팅 옵션:
   - **폴드**: 패를 버리고 라운드 포기
   - **체크**: 베팅 없이 넘김 (베팅이 없을 때만)
   - **콜**: 상대방의 베팅과 같은 금액 베팅
   - **레이즈**: 베팅 금액을 올림 (슬라이더로 조절)
4. 베팅 라운드: 프리플랍 → 플랍(3장) → 턴(1장) → 리버(1장) → 쇼다운
5. 가장 강한 5장의 카드 조합으로 승부

## 핸드 랭킹 (강한 순서)

1. **스트레이트 플러시**: 같은 무늬의 연속된 5장
2. **포카드**: 같은 숫자 4장
3. **풀하우스**: 트리플 + 원페어
4. **플러시**: 같은 무늬 5장
5. **스트레이트**: 연속된 숫자 5장
6. **트리플**: 같은 숫자 3장
7. **투페어**: 같은 숫자 2장씩 2쌍
8. **원페어**: 같은 숫자 2장
9. **하이카드**: 가장 높은 카드

## 실행 방법

`index.html` 파일을 웹 브라우저로 열면 됩니다.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript

## 🎲 블랙잭 게임 규칙

- **시작 칩**: 1000칩으로 시작
- **승리 조건 선택**: 
  - 1,500 칩 / 2,000 칩 / 5,000 칩 / 10,000 칩
  - 무제한 모드 (목표 없이 플레이)
- **목표**: 딜러보다 21에 가깝게 만들되, 21을 넘으면 안됨
- **카드 값**: 
  - 숫자 카드: 그대로 계산
  - J, Q, K: 10으로 계산
  - A: 1 또는 11 (자동 계산)
- **딜러 규칙**: 17 이상이면 스탠드, 16 이하면 히트
- **블랙잭**: 처음 2장으로 21을 만들면 1.5배 지급
- **최종 승리**: 목표 칩을 달성하면 게임 승리!
- **패배**: 모든 칩을 잃으면 게임 종료

## 🎮 블랙잭 게임 방법

1. "블랙잭" 버튼 클릭
2. **승리 조건 선택**:
   - 1,500 / 2,000 / 5,000 / 10,000 칩
   - 무제한 모드
3. 베팅 칩 선택 (10, 25, 50, 100, 200)
4. "딜" 버튼으로 카드 받기
5. 플레이 옵션:
   - **히트**: 카드 한 장 더 받기
   - **스탠드**: 더 이상 카드를 받지 않음
   - **더블 다운**: 베팅을 2배로 올리고 카드 1장만 받고 스탠드
6. 딜러가 자동으로 플레이
7. 점수 비교로 승부 결정
8. 목표 칩 달성 시 게임 승리! 🎉

## 💎 슬롯머신 게임 규칙

- **시작 칩**: 1000칩으로 시작
- **승리 조건 선택**: 
  - 1,500 칩 / 2,000 칩 / 5,000 칩 / 10,000 칩
  - 무제한 모드
- **테마**: 다이아몬드 💎
- **심볼 & 배당**:
  - 💎 (큰 다이아) × 3 = **50배** 🔥
  - 💎💎💎 (작은 다이아3) × 3 = **30배**
  - 💎💎 (작은 다이아2) × 3 = **20배**
  - 7️⃣ (럭키세븐) × 3 = **25배**
  - 🪙 (황금동전) × 3 = **15배**
  - 🍒 (체리) × 3 = **10배**
  - 🍍 (파인애플) × 3 = **5배**
- **베팅**: 10 / 20 / 50 / 100 칩 선택 가능
- **승리**: 3개의 심볼이 모두 일치하면 배당 지급!

## 🎮 슬롯머신 게임 방법

1. "슬롯머신" 버튼 클릭
2. **승리 조건 선택**
3. 베팅 금액 선택 (10 / 20 / 50 / 100)
4. "스핀!" 버튼 클릭
5. 3개의 릴이 돌아가며 심볼 표시
6. 3개가 일치하면 배당 획득! 💰
7. 목표 칩 달성 시 게임 승리! 🎉
